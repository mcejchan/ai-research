## Practical Tips & Procedures

### Setting Up the Newsletter Researcher
**What it does:** Automates the process of researching and drafting newsletters.  
**Steps:**
1. Create a folder on your computer for your newsletter.
2. Inside that folder, create a markdown file called "newsletter links" and add relevant URLs.
3. Open Claude Code and paste in a prompt to set up the slash command for the newsletter researcher.
**Quote:** "I wanted to find a way... to come up with like my own AI employee that does that."  
**When to use:** Use this setup when you need to streamline your newsletter creation process and save time on research.

### Daily Brief Command
**What it does:** Provides a summary of the latest news in areas of interest.  
**Steps:**
1. Create a slash command in Claude Code that searches the web for news related to your interests (e.g., AI, tech).
2. Set it to compile a daily brief with publication dates, sources, and key insights.
**Quote:** "This actually will go search the web... and build me priority updates."  
**When to use:** Use this command to stay updated on industry trends without manually searching for news.
