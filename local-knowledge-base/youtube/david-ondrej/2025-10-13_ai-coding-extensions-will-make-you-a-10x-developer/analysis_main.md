## Practical Tips & Procedures

### Setting Up Client
**What it does:** Installs the Client extension for AI coding assistance.
**Steps:**
1. Open your code editor (e.g., VS Code) and navigate to the extensions tab.
2. Search for "Client" and click install.
3. Open the primary sidebar and configure your API settings.
4. Go to API configuration, select Open Router, and paste your API key.
5. Choose your desired model (e.g., GBT5) and enable necessary settings.
**Quote:** "The setup is simple but it's also not that customizable."  
**When to use:** Use this setup for basic AI coding assistance and when starting with AI tools.

### Using Open Router
**What it does:** Allows access to multiple AI models through a single API.
**Steps:**
1. Create an account on openrouter.ai.
2. Charge your account with a small amount (e.g., $5).
3. Generate an API key and copy it.
4. In your extension settings, select Open Router as your provider and paste the API key.
5. Choose the model you want to use and configure settings as needed.
**Quote:** "This allows you to use all of the models."  
**When to use:** When you want flexibility in choosing AI models without being locked into a single provider.
