## Practical Tips & Procedures

### Setting Up OpenClaw on Hostinger
**What it does:** Guides users through deploying OpenClaw on a VPS using Hostinger.  
**Steps:**
1. Go to the Hostinger landing page and click on deploy.
2. Select a 24-month plan for discounts and enter the coupon code "David" for an additional 10% off.
3. Register for a Hostinger account and log in.
4. Copy the OpenClaw gateway token and select the AI model (Enthropic).
5. Obtain the Enthropic API key and paste it into the Hostinger panel.
6. Click deploy to finalize the setup.  
**Quote:** "Literally couldn't be simpler. We didn't touch the terminal at all."  
**When to use:** Use this setup process when starting with OpenClaw to ensure a smooth and efficient deployment.

### Configuring Heartbeat for Proactive Updates
**What it does:** Sets up OpenClaw to check for updates and improve security.  
**Steps:**
1. Check the current configuration of the heartbeat in OpenClaw.
2. Browse the web for instructions on configuring the heartbeat.
3. Update the heartbeat.md file with prompts to check for official updates every 10 minutes.  
**Quote:** "One of the best ways to protect yourself in any software is to have the latest update."
