# The future of agentic coding with Claude Code
**Channel:** Anthropic  
**URL:** https://www.youtube.com/watch?v=iF9iV4xponk

## Key Takeaways

### 1. Evolution of Coding with AI
**Quote:** "A year ago, if you want to write code, you have an IDE, you have some sort of autocomplete in the IDE... now when you code, you use an agent, you don't directly manipulate text in an IDE anymore."  
**Why it matters:** This shift signifies a major transformation in software development, where AI agents are becoming integral to the coding process, allowing developers to focus on higher-level tasks rather than manual text manipulation.

### 2. The Importance of Feedback Loops
**Quote:** "Whenever someone gave feedback, I would try to fix it as fast as I can... the Claude Code feedback channel internally is just this fire hose, just nonstop."  
**Why it matters:** Establishing a responsive feedback mechanism is crucial for product improvement. It encourages user engagement and ensures that the development team can quickly address issues and enhance the product based on real user experiences.

### 3. Preparing for the Future of Coding
**Quote:** "You still have to learn the craft... but also just start to get more creative."  
**Why it matters:** As coding becomes more accessible through AI tools, developers need to balance technical skills with creativity. This shift emphasizes the importance of innovative thinking in addition to traditional coding knowledge.

## Practical Tips & Procedures

### Getting Started with Claude Code
**What it does:** Helps users familiarize themselves with the codebase before writing code.  
**Steps:**
1. Ask Claude Code questions about the codebase, such as how to implement new features or understand existing functions.
2. Explore the codebase through Claude Code's insights before attempting to write code.
3. Once comfortable, transition to using Claude Code for coding tasks.
**Quote:** "Don't use it to write code yet... ask Claude Code questions about the code base."  
**When to use:** Ideal for new users who need to understand their codebase before diving into coding.

### Task Categorization for Effective Use
**What it does:** Guides users on how to approach tasks based on complexity.  
**Steps:**
1. Identify the task's complexity: categorize it as easy, medium, or hard.
2. For easy tasks, use Claude Code to generate the code directly via GitHub tagging.
3. For medium tasks, start with a planning phase in Claude Code before implementation.
4. For hard tasks, take the lead while using Claude Code as a supportive tool for research and prototyping.
**Quote:** "Think about what's the task that you're doing and what's the right way to use Claude Code to do it."  
**When to use:** Useful for developers at any level to optimize their workflow with Claude Code.
